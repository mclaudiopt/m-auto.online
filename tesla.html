📚 Guias Rápidos

Tesla Toolbox 3: Software oficial Tesla para diagnóstico e serviço. Requer subscrição Tesla (diária/mensal/anual) e conta Tesla Service.

Service Mode: Modo de diagnóstico integrado no ecrã do Tesla. Acede a sensores, câmaras, HVAC, bateria e mais.

SEXY Buttons / Commander: Dispositivo aftermarket para comandos físicos e acesso a funções de diagnóstico via CAN bus.

Comandos de Serviço: Procedimentos de calibração, reset e diagnóstico acessíveis via Service Mode ou Toolbox.

Adaptador Ethernet/DoIP: Tesla usa Ethernet para diagnóstico. Adaptador USB-Ethernet necessário para Toolbox 3.
