¶ 获取安装程序

使用设备开始汽车诊断之前，PC 上必须安装 VX Manager 管理工具和驱动程序，此安装程序包含在产品 CD-ROM 中，或者您可通过以下链接下载最新版本安装程序：
http://www.allscanner.com
http://www.vxdiag.net/#download

¶ 软件安装需求

  • 处理器： 1.6 GHz 或更快。
  • 内存： DDR 4GB 或以上。
  • 硬盘： 80GB 或以上。
  • 网络接口： LAN 100/1000M。
  • 通信接口： USB2.0 或 USB3.0。
  • 无线网络： 802.11a/b/g/n WiFi。
  • 操作系统： Windows 10 / 8 / 7。
  • 浏览器： Internet Explorer 11 或更新版本。

DoNet 远程诊断组件需要使用 Internet Explorer 11 或更新版本。
Windows 7 系统需要升级 IE 浏览器为最新版本。
下载 Internet Explorer 11（32 位）
下载 Internet Explorer 11（64 位）

注意：Windows XP 系统已经停止支持！

¶ 软件安装步骤

安装过程中请关闭 Windows Defender 或其他安全/杀毒软件！

¶ 1. 运行安装程序

双击运行 VX Manager 安装程序。

¶ 2. 开始安装

欢迎安装界面，点击【Next】继续。

¶ 3. 选择安装组件

此过程中可以勾选要安装的原厂诊断驱动，或者安装完成后使用 VX Mananger 自由安装需要的原厂驱动。

¶ 4. 安装进行中

安装过程可能持续数分钟，请耐心等待。

¶ 5. 安装完成

安装完成界面，点击【Finish】完成。

安装完成后将在桌面和开始菜单生成 VX Manager 快捷方式。

¶ 连接设备和车辆

以 VCX-DoIP 设备通过USB连接为例，请确保设备连接正常：

  • 使用 DB26-OBD 电缆连接设备至车辆。
  • 使用 USB Type-B 电缆连接设备至 PC。

  • 更多连接方式 详细了解 VCX 系列多种设备不同连接方式的使用

¶ 基本功能

  • 【重新连接】 ── 重新连接设备并刷新设备信息。
  • 【设备测试】 ── 设备自检和 LED 指示灯闪烁并且发出蜂鸣提示音。
  • 【重启设备】 ── 设备复位并重新启动运行。
  • 【更新固件】 ── 在线下载并更新设备固件程序。
  • 【更新授权】 ── 在线下载并更新设备授权数据。
  • 【DoIP开关】 ── 激活或关闭DoIP协议，测试车辆DoIP通信。
  • 【解除占用】 ── 设备被占用后手动解除占用释放设备。
  • 【错误报告】 ── 查看和获取设备错误日志。
  • 【检查更新】 ── 检查 VX Manager 软件是否有更新。

¶ 安装原厂诊断驱动

打开 【车辆诊断】->【我的应用】 页面，点击要安装的诊断应用 [JLR DoIP]，在驱动信息界面中点击 【安装】 完成驱动安装。
