¶ 设备 IP 地址修改

VCX-DoIP 设备连接车辆后相当于一个车辆网关设备，设备本身的网关 IP 地址默认为 192.168.8.1。
在 DoNet 远程使用过程中设备需要连接到用户路由器，某些情况下路由器使用的 IP 地址与 VCX 设备可能相同产生冲突，此时可以修改 VCX 设备的网关 IP。本文档介绍修改设备 IP 地址的方法。

请先连正确连接设备和电脑。

¶ 1. 登陆设备管理页面

使用浏览器打开设备管理页面 192.168.8.1 并登录

¶ 2. 打开网络设置

进入 Network -> Interface, 点击 LAN 的 [Edit]

¶ 3. 修改网络 IP

修改 IPv4 address, 点击 [Save & Apply]

¶ 4. 修改 VXManager 配置

设备 IP 地址修改成功后，必须同时修改 VX Manager 客户端管理工具的连接配置，打开【设备配置】选择 [LAN] 同时修改 IP地址和设备IP一致，点击【保存】

使用其他连接方式如 WLAN / USBLAN 时也需要修改对应的 IP 地址。
