¶ 产品概述

VCX-FD 是傲世卡尔设计的智能车辆诊断接口，代表着下一代的诊断技术。产品采用全新升级的双核高速处理器，全面支持多通道 CAN-FD / DoIP 和传统诊断协议，并且兼容更多原厂诊断软件。VCX-FD 还支持本地和超级远程双模式诊断，为用户提供卓越的诊断体验。

¶ 产品特性

¶ 极致性能的车辆诊断接口

¶ 双核架构协同工作，通信性能更强

  • 550MHz ARM 协议处理器实现 CAN-FD 和所有传统车辆总线协议。
  • 560MHz MIPS 网络处理器实现 DoIP 超级诊断网关。
  • 双核处理器通过百兆以太网通信，高带宽、低延迟。
  • 诊断通信速度比上一代产品提升数倍。

¶ 双操作系统并发，软件运行更稳定

  • RTOS 实时操作系统保障车辆总线协议多任务高并发通信。
  • OpenWrt 网络操作系统灵活连接多种网络接口和车辆。

¶ 支持更全的车辆总线协议

  • 集成 OBD 智能协议多路器芯片，所有引脚能够智能切换任意协议。
  • 支持 3 路 CAN-FD 多通道并发通信，最高支持 5M Bps 高速通信。
  • 支持 ISO-13400 标准双路 DoIP 接口 Option 1 和 Option 2。

¶ 灵活易用的连接方式

  • 主机通信接口支持 USB / RJ45 / WLAN 等多种连接, 灵活易用。
  • 内置无线路由模式，诊断电脑可无线直连设备。
  • 支持无线工作站模式，设备可接入无线局域网，任意电脑都可连接设备。
  • 支持将车辆 DoIP 以太网映射到 WLAN 实现无线 DoIP 诊断。

¶ 全新的外观设计

  • 采用集成 OBD-II 接口的一体化设计，方便即插即用。
  • 加固设计的 Type-C 接口可防止晃动和脱落，连接更可靠。

¶ 更强大的车辆诊断设备

¶ 更全面的原厂车辆诊断

  • VCX-FD 现在适配的原厂诊断车型品牌已达 17 种。
  • 硬件更新后全面支持基于 CAN-FD 和 DoIP 的新车型和新原厂诊断软件。
  • 各种高级诊断和编程功能达到原厂级别，速度甚至超过原厂设备。
  • 支持一键安装、卸载和升级原厂支持驱动，易于使用。

¶ 更新的国际标准 API 接口

  • SAE-J2534-1/2 PassThru V04.04 & V05.00
    • SAE-J2534 Extension for GMW17753 (GM CAN FD)
  • ISO-22900 D-PDU API for MVCI
  • RP-1210A/B/C API for VDA
  • 可提共全平台 API 接口 (Windows / Linux / Android)

¶ J2534 原厂级 ECU 编程功能

  • ECU 软件升级和标定。
  • ECU 更换刷写和编程。
  • J2534 可控编程电压 (5~20V) 输出。

¶ 双模式智能远程诊断

  • 超级远程诊断只需设备联网即可实现原厂在线诊断。
  • 兼容远程诊断可支持多种特殊功能设备远程诊断。
  • 设备支持一键配网功能，快速接入远程诊断平台。
