Change Device IP Address

VCX-DoIP 设备Connection车辆后相当于一个车辆网关设备，设备本身的网关 IP 地址默认为 192.168.8.1。
在 DoNet 远程使用过程中设备需要Connection到用户路由器，某些情况下路由器使用的 IP 地址与 VCX 设备可能相同产生冲突，此时可以修改 VCX 设备的网关 IP。本文档介绍修改设备 IP 地址的方法。

请先连正确to connect the device to the PC.

1. 登陆设备管理页面

使用浏览器Open设备管理页面 192.168.8.1 并登录

2. Open网络设置

进入 Network -> Interface, Click LAN 的 [Edit]

3. 修改网络 IP

修改 IPv4 address, Click [Save & Apply]

4. 修改 VXManager 配置

Change Device IP Address成功后，必须同时修改 VX Manager 客户端管理工具的Connection配置，Open[设备配置]Select [LAN] 同时修改 IP地址和设备IP一致，Click[Save]

使用其他Connection方式如 WLAN / USBLAN 时也需要修改对应的 IP 地址。
