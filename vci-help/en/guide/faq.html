1.VX Manager Startup Error — ErrorCode = 1450

Problem description:
VX Mananger Installation Complete后启动报错：”ERROR: Loadlibrary(VCX.dll) ErrorCode = 1450“。

Solution
Uninstall and reinstall VX Manager.

2.Device Connected but VX Manager Does Not Detect It

问题描述
设备USB连上电脑了 但是客户端VX MANAGER找不到设备，Open网络适配器页面发现有设备Connection，而且设备IP正确（192.168.8.* **）

Step 1: Check that the network adapter detects the device

Step 2: Confirm device IP is correct (192.168.8.*) — select "Obtain IP automatically"

Solution：

Fix 1: Change the connection mode to USBLAN.

②.如果客户端设置为USBLAN还是没连上设备，请Open计算机设备管理，看看网络适配器里设备网卡驱动是否异常（如图），异常请卸载该网卡后在设备管理内Refresh就可以了。

③.如果网卡，客户端和设备管理都没找到设备，请看电脑的WIFIConnection是否有设备的WIFI信号“DOIP-VCI-****”,没有的话设备返修。

3.Mercedes-Benz WIS Registration Fails

Problem description:When trying to register WIS, there is no Hardware ID.

Solution：这是WIS已经在别的电脑注册过，或者拷贝硬盘不完整，重拷硬盘。

4. GM GDS2 No License Date After Installation
