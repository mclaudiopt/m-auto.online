Get the Installer

Before starting vehicle diagnosis, the PC must have VX Manager and drivers installed. The installer is included in the product CD-ROM or can be downloaded from these links:
http://www.allscanner.com
http://www.vxdiag.net/#download

Software Requirements

  • CPU: 1.6 GHz or faster.
  • RAM: DDR 4 GB or more.
  • HDD: 80 GB or more.
  • Network interface: LAN 100/1000 Mbps.
  • Communication interface: USB 2.0 or USB 3.0.
  • Wireless: 802.11a/b/g/n WiFi.
  • OS: Windows 10 / 8 / 7.
  • Browser: Internet Explorer 11 or newer.

The DoNet Remote Diagnosis component requires Internet Explorer 11 or newer.
Windows 7 requires upgrading IE to the latest version.
Download Internet Explorer 11 (32-bit)
Download Internet Explorer 11 (64-bit)

Note: Windows XP is no longer supported!

Installation Steps

Disable Windows Defender or other antivirus software during installation!

1. Run the Installer

Double-click to run the VX Manager installer.

2. Begin Installation

Welcome screen — click [Next] to continue.

3. Select Components

You can select which OEM diagnostic drivers to install，或者Installation Complete后使用 VX Mananger 自由安装需要的原厂驱动。

4. Installation in Progress

The installation may take several minutes. Please wait.

5. Installation Complete

Completion screen — click [Finish] to close.

After installation, a VX Manager shortcut is created on the desktop and Start menu.

Connect Device and Vehicle

Using VCX-DoIP connected via USB as an example, ensure the connection is correct:

  • Connect the DB26-OBD cable from the device to the vehicle.
  • Connect the USB Type-B cable from the device to the PC.

  • More connection methods Learn about different connection methods for VCX series devices

Using the VCX Manager Tool

Launch VX Manager. If the device is connected successfully, the following information will be displayed:

Basic Functions

  • [Reconnect] ── Reconnect device and refresh device information.
  • [Device Test] ── Device self-test with LED flash and beep.
  • [Restart Device] ── Reset and restart the device.
  • [Update Firmware] ── Download and update device firmware online.
  • [Update License] ── Download and update device license data online.
  • [DoIP Toggle] ── Activate or deactivate DoIP protocol to test vehicle DoIP communication.
  • [Release Device] ── 设备被占用后手动Release Device释放设备。
  • [Error Report] ── View and retrieve device error log.
  • [Check for Updates] ── Check if VX Manager has updates available.
