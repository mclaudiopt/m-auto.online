固件更新

VCX 设备通过Update Firmware来不断优化设备和新增Features，请保持设备固件始终为最新！

Connection设备

For firmware updates, wired connection (USB or LAN) is recommended. See Device Connection Setup to connect the device to the PC.

USB connection diagram:

When unable to connect to a vehicle, the device can be powered via USB.

Some PC USB ports may not supply sufficient power, causing instability. Power the device via the OBD port instead.

查看固件Version

Launch VX Manager. The [设备信息] section shows the current firmware version. If new firmware is available on the server, [Update Firmware] will show [+] 标记，并且自动弹出Update Firmware对话框。

Update Firmware过程

在Update Firmware对话框中勾选需要更新的固件，单击 [Start更新]。This takes a few minutes. Do not disconnect the device during the update.

更新Done

After closing the update dialog, VX Manager [设备信息] will display the updated firmware version.

DoIP 固件恢复

For DoIP system boot failures caused by abnormal conditions, try recovering the DoIP firmware.

This procedure only applies to devices with a DoIP system (VCX-DoIP, VCX-SE).

Criteria for detecting DoIP system abnormality:
设备启动25秒后，没有听到第二次蜂鸣器 “哔哔” 声。
After 25 seconds, the wireless LED does not light or flash.
After 25 seconds, the device wireless network is not found: DoIP-VCI-XXXX。
After 25 seconds, VX Manager cannot connect or read device information.
