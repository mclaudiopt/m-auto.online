Firmware Update

M-Auto VCI The device viaUpdate Firmwareto continuously optimize and add featuresFeatures，keep device firmware always updated！

ConnectionDevice

For firmware updates, wired connection (USB or LAN) is recommended. See Device Connection Setup to connect the device to the PC.

USB connection diagram:

When unable to connect to a vehicle, the device can be powered via USB.

Some PC USB ports may not supply sufficient power, causing instability. Power the device via the OBD port instead.

checkFirmwareVersion

Launch VCI Manager. The [DeviceInformation] section shows the current firmware version. If new firmware is available on the server, [Update Firmware] will show [+] mark，andandAutomaticpop-upUpdate Firmwareformeansdialog。

Update Firmwareprocess

inUpdate Firmwareformeansdialogcentercheck/selectneedneedUpdateFirmware，singleclick [StartUpdate]。This takes a few minutes. Do not disconnect the device during the update.

UpdateDone

After closing the update dialog, VCI Manager [DeviceInformation] will display the updated firmware version.

DoIP Firmwarerestore

For DoIP system boot failures caused by abnormal conditions, try recovering the DoIP firmware.

This procedure only applies to devices with a DoIP system (M-Auto VCI DoIP, M-Auto VCI SE).

Criteria for detecting DoIP system abnormality:
Devicestart25secondsafter，withoutheartosecond beep “beep” sound。
After 25 seconds, the wireless LED does not light or flash.
After 25 seconds, the device wireless network is not found: DoIP-VCI-XXXX。
After 25 seconds, VCI Manager cannot connect or read device information.
