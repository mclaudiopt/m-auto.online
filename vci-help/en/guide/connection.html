WLAN Connection方式适用产品：VCX-DOIP / VCX-SE

VCX-SE WLAN Connection示意

WLAN ConnectionStep

  • 电脑搜寻Wireless:DoIP-VCI-XXXX，选中 自动Connection，然后Click Connection。

  • EnterWireless网络密码：12345678，Click Next。

USBLAN - USB 网卡

USBLAN Connection方式适用产品：VCX-DOIP / VCX-SE

VCX-DoIP USBLAN Connection示意

VCX-SE USBLAN Connection示意

首次Connection

  • [首次Connection USBLAN 时，Windows 会自动安装驱动…]

USB (VCX-NANO)

USB Connection方式只适用于 VCX-NANO 产品！

USB VCX-NANO Connection示意

首次Connection

  • [首次Connection USB 串口时， Windows 会自动安装驱动…]


  • Driver Installation成功后可在设备管理器中看到端口：USB Serial Port (COMx)。

WiFi (VCX-NANO)

此WirelessConnection方式只适用于带WirelessFeatures的 VCX-NANO 产品！

WiFi VCX-NANO Connection示意

WiFi ConnectionStep

  • [设备Connection启动正常。…]
  • 电脑搜寻Wireless:VCX-WiFi，Click [Connection]。
  • [无需Enter密码，Connection成功。…]

设备Connection检测

设备按照以上方法正确Connection后，Launch VX Manager. The如能正确显示 VCX 设备的Serial number、Version、名称等信息，则说明设备已正确Connection上电脑

Device Connection Setup

VX Manager 默认Connection类型配置为: AUTO自动， 可以Support自动识别所有Connection方式，一般无需修改。
如果您Confirm需要修改Connection类型，可在 设备管理 -> 设备配置 界面中修改。
