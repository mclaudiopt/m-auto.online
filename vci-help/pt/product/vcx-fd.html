Descrição do Produto

O VCX-FD é a interface de diagnóstico veicular inteligente da ALLScanner, representando a tecnologia de diagnóstico de próxima geração. Equipado com um novo processador dual-core de alta velocidade, suporta totalmente CAN-FD/DoIP multi-canal e protocolos de diagnóstico tradicionais, com compatibilidade mais ampla com software OEM. O VCX-FD suporta diagnóstico local e super remoto de modo duplo.

Características do Produto

Interface de Diagnóstico Veicular de Alto Desempenho

Arquitectura Dual-Core, Comunicação Mais Potente

  • [Processador ARM 550 MHz para CAN-FD e todos os protocolos de barramento veicular.…]
  • [Processador de rede MIPS 560 MHz para gateway de super diagnóstico DoIP.…]
  • [Os dois processadores comunicam via ethernet 100 Mbps — alta largura de banda, baixa latência.…]
  • [Velocidade de comunicação de diagnóstico várias vezes superior à geração anterior.…]

Dois Sistemas Operativos em Paralelo, Maior Estabilidade

  • [O RTOS garante comunicação multi-tarefa e alta concorrência nos protocolos de barramento.…]
  • [O OpenWrt permite ligação flexível a múltiplas interfaces de rede e veículos.…]

Suporte Completo de Protocolos de Barramento Veicular

  • [Chip multiplexador de protocolo OBD inteligente — todos os pinos podem comutar para qualquer protocolo.…]
  • [Suporta 3 canais CAN-FD simultâneos até 5 Mbps.…]
  • [Suporta interfaces DoIP duplas ISO-13400 Opção 1 e Opção 2.…]

Modos de Ligação Flexíveis

  • [Interface de comunicação suporta USB/RJ45/WLAN e outros modos.…]
  • [Modo router sem fios integrado — o PC de diagnóstico pode ligar-se directamente ao dispositivo sem fios.…]
  • [Suporta modo estação wireless — o dispositivo pode ser acedido por qualquer PC na rede local.…]
  • [Suporta mapeamento da ethernet DoIP do veículo para WLAN, permitindo diagnóstico DoIP sem fios.…]

Design Exterior Renovado

  • [Design integrado com conector OBD-II para fácil ligação plug-and-play.…]
  • [Conector Type-C reforçado para evitar folgas, ligação mais fiável.…]

Dispositivo de Diagnóstico Veicular Mais Potente

Diagnóstico OEM Mais Abrangente

  • [O VCX-FD é compatível com 17 marcas de diagnóstico OEM.…]
  • [O hardware actualizado suporta totalmente novos modelos e software OEM baseados em CAN-FD e DoIP.…]
  • [Funções avançadas de diagnóstico e programação ao nível OEM, com velocidade que supera os equipamentos originais.…]
  • [Suporta instalação, desinstalação e actualização de drivers OEM com um clique.…]

Interfaces API de Normas Internacionais Actualizadas

  • SAE-J2534-1/2 PassThru V04.04 & V05.00
    • SAE-J2534 Extension for GMW17753 (GM CAN FD)
  • ISO-22900 D-PDU API for MVCI
  • RP-1210A/B/C API for VDA
  • [Fornece API multiplataforma (Windows / Linux / Android)…]

Programação ECU de Nível OEM com J2534

  • [Actualização e calibração de software ECU.…]
  • [Substituição, reprogramação e programação de ECU.…]
  • [Tensão de programação controlável J2534 (5~20V).…]

Diagnóstico Remoto Inteligente de Modo Duplo

  • [O Super Diagnóstico Remoto apenas requer ligação à Internet para diagnóstico OEM online.…]
  • [兼容远程诊断可Suporte多种特殊Funções设备远程诊断。…]
  • [设备Suporte一键配网Funções，快速接入远程诊断平台。…]
