固件更新

VCX 设备通过Actualizar Firmware来不断优化设备和新增Funções，请保持设备固件始终为最新！

Ligação设备

Para actualização de firmware recomenda-se ligação com fio (USB ou LAN). Consulte Configuração de Ligação do Dispositivo para ligar o dispositivo ao computador.

Diagrama de ligação USB:

Quando não for possível ligar ao veículo, o dispositivo pode ser alimentado via USB.

Algumas portas USB de PC podem não fornecer potência suficiente. Nesse caso, alimente o dispositivo pela porta OBD.

查看固件Versão

Inicie o VX Manager. A secção [Informações do Dispositivo] mostra a versão actual do firmware. Se o servidor contiver novo firmware, [Actualizar Firmware] mostrará [+] 标记，并且自动弹出Actualizar Firmware对话框。

Actualizar Firmware过程

在Actualizar Firmware对话框中勾选需要更新的固件，单击 [Iniciar Actualização]。O processo demora alguns minutos. Não interrompa a ligação durante a actualização.

更新Concluído

Após fechar a janela de actualização, o VX Manager [Informações do Dispositivo] mostrará a versão de firmware actualizada.

DoIP 固件恢复

Para falhas de arranque do sistema DoIP por anomalias, tente recuperar o firmware DoIP.

Este processo aplica-se apenas a dispositivos com sistema DoIP (VCX-DoIP, VCX-SE).

Critérios para detectar anomalia no sistema DoIP:
设备启动25秒后，没有听到第二次蜂鸣器 “哔哔” 声。
Após 25 segundos, o LED sem fios não acende nem pisca.
Após 25 segundos, a rede sem fios do dispositivo não é encontrada: DoIP-VCI-XXXX。
Após 25 segundos, o VX Manager não consegue ligar ou ler as informações do dispositivo.
