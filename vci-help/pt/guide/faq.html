1.Erro de Arranque do VX Manager — ErrorCode = 1450

Descrição:
VX Mananger Instalação Concluída后启动报错：”ERROR: Loadlibrary(VCX.dll) ErrorCode = 1450“。

Solução
Desinstale e reinstale o VX Manager.

2.O Dispositivo Está Ligado mas o VX Manager Não o Detecta

问题描述
设备USB连上电脑了 但是客户端VX MANAGER找不到设备，Abra网络适配器页面发现有设备Ligação，而且设备IP正确（192.168.8.* **）

Passo 1: Verificar que o adaptador de rede detecta o dispositivo

Passo 2: Confirmar que o IP do dispositivo está correcto (192.168.8.*) — seleccionar "Obter IP automaticamente"

Solução：

Solução 1: Altere o modo de ligação para USBLAN.

②.如果客户端设置为USBLAN还是没连上设备，请Abra计算机设备管理，看看网络适配器里设备网卡驱动是否异常（如图），异常请卸载该网卡后在设备管理内Actualize就可以了。

③.如果网卡，客户端和设备管理都没找到设备，请看电脑的WIFILigação是否有设备的WIFI信号“DOIP-VCI-****”,没有的话设备返修。

3.Mercedes-Benz WIS Não Regista

Descrição:Ao tentar registar o WIS, não há Hardware ID.

Solução：这是WIS已经在别的电脑注册过，或者拷贝硬盘不完整，重拷硬盘。

4. GM GDS2 Sem Data de Licença Após Instalação

Descrição:GM GDS2 Sem Data de Licença Após Instalação；找不到设备；测不了车。

Solução：请让客户查看他的系统，该软件的破解不Suporte家庭Versão的WINDOWS！Windows home.Instale numa versão diferente. O Tech2win apenas suportaWin7 32位para instalação e utilização.
