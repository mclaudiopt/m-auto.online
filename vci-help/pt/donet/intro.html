Introdução ao Diagnóstico Remoto

DoNet Diagnóstico Remoto，é baseado em computação em nuvem e5GComunicaçãoplataforma de diagnóstico remoto em tempo real。Os utilizadores usam M-Auto VCI dispositivo de diagnóstico，através de DoNet WeChatsinalMachopúbliconúmeroPlataforma，entãoalcançarfalhaVeículo(Cliente C) comempresa de reparaçãoouespecialistas técnicos(Servidor S)interligação。

Através desta plataforma, oficinas e clientes finais sem competências técnicas ou equipamentos adequados podem pedir assistência remota. Os especialistas podem oferecer diagnóstico, codificação e programação a qualquer cliente, sem limitações geográficas.

Diagnóstico Remotogrupocomposto

ClienteeServidor

O cliente (C) é a parte que necessita de diagnóstico remoto para um veículo avariado.

O servidor (S) é a parte que fornece o serviço de diagnóstico remoto.

M-Auto VCI SE Diagnóstico RemotoDispositivo

O M-Auto VCI SE é o dispositivo de diagnóstico de última geração da M-Auto para a plataforma de diagnóstico remoto.


Design de hardware de alta integração com suporte KWP/CAN/DoIP, interfaces WiFi/USB/LAN e drivers J2534/PDU para diagnóstico OEM. Suporta diagnóstico local e ligação rápida à plataforma remota com um clique.
O servidor (S) e o cliente (C) podem usar o mesmo dispositivo M-Auto VCI SE. Os dispositivos M-Auto VCI DoIP existentes também podem ser activados para diagnóstico remoto.

DoNet Diagnóstico RemotoWeChatsinalMachopúbliconúmero


O canal WeChat DoNet (DiagnosisOnNet) é uma plataforma online da M-Auto para oficinas, especialistas e clientes. Após seguir o canal, pode associar o M-Auto VCI, configurar a rede e verificar o estado; clientes (C) publicam pedidos e servidores (S) aceitam serviços de diagnóstico.

DuploModoDiagnóstico Remoto

O DoNet suporta de forma inovadora dois modos de diagnóstico remoto: Super Diagnóstico Remoto e Diagnóstico Remoto Compatível.

Super Diagnóstico Remoto

O modo Super apenas requer um dispositivo M-Auto VCI no cliente para diagnóstico remoto multi-marca com software OEM directo.
O cliente (C) liga o dispositivo M-Auto VCI ao veículo avariado e configura a rede via WeChat.
O servidor (S) não precisa de dispositivo. Com PC e software OEM, carrega drivers PDU/J2534 remotos para aceder directamente ao veículo remoto e prestar serviços OEM.

Super Diagnóstico RemotoCaracterísticas

  • [Suporta ~20 marcas: Mercedes-Benz, BMW, Porsche, VW, Audi, Land Rover, GM, Ford, Toyota, Honda, Subaru, Volvo, entre outras.…]
  • [Suporte profundo do software OEM — o diagnóstico remoto oferece as mesmas capacidades que o local.…]
  • [Suporta diagnóstico remoto via DoIP, com ethernet do veículo mapeada directamente para o servidor.…]
  • [O servidor apenas precisa de conta e certificados para codificação e programação remota em qualquer cliente.…]

Diagnóstico Remoto Compatível

O modo Compatível usa 1 dispositivo M-Auto VCI em cada lado para diagnóstico remoto com qualquer marca.
O cliente (C) liga o M-Auto VCI ao veículo, configura a rede e envia o pedido de diagnóstico via WeChat.
O servidor (S) alimenta o M-Auto VCI via adaptador OBD e configura a rede. Pode usar qualquer dispositivo OEM ou de terceiros via OBD para prestar o serviço.

Diagnóstico Remoto CompatívelCaracterísticas

  • [O servidor pode usar qualquer dispositivo: OEM, scanner genérico, ferramentas anti-roubo, TPMS, tuning, etc.…]
  • [O diagnóstico remoto compatível suporta actualmente CAN e DoIP.…]
  • [Suporta todos os modelos que usam CAN e DoIP.…]
